# AI-Based Battery Thermal Optimization using CFD and Machine Learning

## Overview

This project combines Computational Fluid Dynamics (CFD) and Machine Learning (ML) to optimize battery thermal management systems.

Thermal simulations were performed in ANSYS Fluent to study the effects of:

* Airflow velocity
* Heat generation
* Inlet temperature

on:

* Maximum battery temperature
* Average battery temperature
* Pressure drop

The generated CFD dataset was then used to train a Random Forest Machine Learning model for predictive thermal analysis and optimization.
